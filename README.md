* Run all tests locally: $ gradle test
* Run all tests in Travis CI (it's automatically configured to use Gradle 4.0.0) : $ ./gradlew test

* Spring 4.3.10
* Spring Boot 1.5.6
* Spock 1.1
* IntelliJ IDEA 2016.1.4
